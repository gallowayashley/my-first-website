How to push website live using GitHub:

  1. Create a new repository
  2. Name repository + add description
  3. Make public or private (private for tutorials/learning)
  4. Create repository
  5. If you can't find where file is located, you can go to Repository > Code > GitHub Desktop > Repository in top menu > Show in Explorer (to find file location)
  6. Commit changes to master (only important files) in GitHub desktop
  7. Then Publish Repsository button at the top
  8. Click on repository name and go to Settings
  9. Pages > GitHub Pages > Source: Main branch
  10. You will see link to webpage here that you can click on. Ta-da!

To edit the file in the GitHub folder:

  1. From GitHub desktop, go to Repository in top menu bar and then Show in Explorer
  2. Open page you need to edit in VSCode
  3. Once you save code and go back to GitHub desktop, you'll see the changes there
  4. Commit button then Push/Publish button at the top
  5. Refresh web browser link and it should show up shortly.
